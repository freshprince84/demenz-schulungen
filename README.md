# Demenz-Schulungen

Schulungsplattform für die Medizinalbranche, spezialisiert auf Demenzbereich und Piktogramme.

## Zweck

Plattform zur Erstellung und Verwaltung von Schulungen für Fachpersonen in der Betreuung und Pflege von Menschen mit Demenz.

## Geplant

- [ ] Konzept und Struktur definieren
- [ ] Piktogramme evaluieren und auswählen
- [ ] Schulungsmodule erstellen
- [ ] Plattform aufbauen
