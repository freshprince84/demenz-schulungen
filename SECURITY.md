# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| main (latest) | yes |

## Reporting a Vulnerability

Bitte melde Sicherheitslücken **nicht** als öffentliches GitHub Issue.

Kontakt: `security@<PROD-DOMAIN>` (Patrick — vor Go-Live eintragen; kann mit `privacy@` identisch sein)

Wir bemühen uns um eine Antwort innerhalb von 7 Tagen.

## Scope

- Demenz-Schulungen Plattform (Next.js App auf Hetzner)
- API-Endpunkte unter `/api/*`

Nicht im Scope: Winston/Clawdbot-Server (separates System).
